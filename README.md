# Unity Movement Labs

A professional learning repository focused on understanding movement systems in Unity through theory, experiments, guided questions, and small gameplay laboratories.

## Purpose

This repository documents a structured exploration of movement programming in Unity 6.3, combining:

- technical theory
- practical Unity labs
- conceptual diagrams
- guided questions and expected answers
- portfolio-oriented documentation

It is designed both as a professional training resource and as a foundation for future teaching material.

## Unity Version

- Unity 6.3
- 2D URP template

## Repository Structure

- `UnityProject/` → Unity project with all practical labs
- `Docs/Theory/` → theoretical foundations
- `Docs/Labs/` → lab guides and tutorials
- `Docs/Questions/` → conceptual questions and answers
- `Docs/Diagrams/` → diagrams used in the documentation
